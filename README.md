# Election_Analysis

## Project Overview
A Colorado Board of Elections employee asked for help in conducting an audit of a recent local congressional elections.
Specifically, he asked for the following tasks to be completed:

1. Calculate the total number of votes cast.
2. Get a complete list of candidates who received votes.
3. Calculate the total number of votes each candidate received.
4. Calculate the percentage of votes each candidate won.
5. Determine the winner of the elections based on popular vote.

## Resources
Data Source: election_results.csv
Software: Python 3.7.6, Visual Studio Code 1.53.2

## Summary

The analysis of the election show that:
- There were 369,711 total votes in the election.
- The candidates were
  - Charles Casper Stockham
  - Diana DeGette
  - Raymon Anthony Doane
- The candidate results were:
  - Charles Casper Stockham received 23.0% of the vote and 85,213 votes
  - Diana Gagette received 73.8% of the vote and 272,892 votes
  - Raymon Anthony Doane received 3.1% of the vote and 11,606 votes
- The winner of the election was:
  - Diana DeGette, who received 73.8% of the vote and 272,892 votes.

## Challenge Overview

In this challenge we were given a csv file with the election results that contained three columns: a ballot ID, the county where the vote took place, and the candidate name chosen in the ballot id.  We were asked to find the total number of candidates and their names, the total number of votes, the total number of votes per candidate, the percent of the vote that each candidate received and finally the winning candidate by popular vote.  

## Challenge Summary

We succeded in conducting the audit and provided all the metrics requested.
